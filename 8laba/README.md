# Отчет по лабораторной работе №8

## Условие задачи
Создать приложение генератор мемов
приложение должно быть написано с применением ОО парадигмы
исключительные ситуации должны обрабатываться с использованием собственных исключений

## Описание проделанной работы

Приложение позволяет быстро создавать мемы из любых изображений. Пользователь загружает картинку, добавляет верхний и нижний текст, а затем сохраняет готовый мем. Программа написана с использованием объектно-ориентированной парадигмы, содержит собственные классы исключений и графический интерфейс на Tkinter
