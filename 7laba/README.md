# Отчет по лабораторной работе №6

## Условие задачи
Сделать программу, которая считает площадь и тепло для комнаты, квартиры или дома. Тепло считается по формуле 100 Вт на 1 м².

## Описание проделанной работы

### Создал абстрактный базовый класс
В файле `models.py` создал абстрактный класс `Geo` с абстрактным методом `area()`. Он задает правило: у всех наследников обязательно должен быть метод для расчета площади.

### Реализовал иерархию наследования
Создал три класса, которые наследуются от `Geo`:
- `Room` – комната с шириной и длиной
- `Apartment` – квартира со списком комнат
- `Building` – дом со списком квартир

### Добавил managed-атрибуты
В классе `Room` сделал свойства `w` и `l` с `@property` и `@setter`. Это позволяет контролировать ввод: ширина и длина не могут быть отрицательными.

### Добавил dunder-методы
- `__str__` – для вывода информации об объекте
- `__len__` – для получения количества комнат или квартир
- `__mul__` – для умножения комнаты на число
- `__getitem__` – для доступа к комнате по индексу

### Сделал расчет площади и тепла
Площадь считается через метод `area()` у каждого объекта. Тепло = `площадь * 100`.

### Сделал интерфейс на PySide6
В файле `main.py` сделал окно с переключателем (комната/квартира/дом), полями для ввода, кнопкой "Рассчитать" и полем для вывода результата.
