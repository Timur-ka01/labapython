## Условие  1 задачи
Создать две функции которые будут считать сумму вложенных список
первая функция будет это делать через цикл
вторая функция будет это делать через рекурсию
## Описанеие проделанной работы 

## Функция P(L)
Создаем функцию которая принимает на вход вложенные списки
задаем переменную result в которой будем считать сумму
задаем переменную L1 которая принимает L и оборачивает его в списко
задаем цикл пока у нас есть возможность брать элементы из нашего списка l1
задаем переменную pos в которую вкладываем крайний элемент нашего списка L1
задаем цикл который проходит по всей переменной pos
если элемент из pos являеться списком то мы добавляем его в конец нашего списка l1, 
тем самым при следующем проходе цикла While переменная pos воззьмет значение следующего вложенного списка
если же элемент из pos являеться числом, то мы прибавляем его к переменной result
возращаем result

## Функция F(l)
Создаем функцию которая принимает на вход вложенный список 
задаем переменную result, где будем считать сумму
задаем цикл в который проходит по элементам вложенного списка
Если элемент является вложенным списком то мы прибавляем к переменной result еще раз нашу функцию, 
потому что наша функция считает сумму в одном списке и поднимает его на уровень выше
если же элемент списка являеться число, то мы просто прибавляем его к результату
возращаем результат
