## задание 0
### уловие:
Дан словарь координат городов.
Необходимо составить словарь словарей, содержащий расстояния между городами.
Расстояние между двумя точками на координатной плоскости вычисляется по формуле:
((x1 - x2) ** 2 + (y1 - y2) ** 2) ** 0.5

### Описание проделанной работы
создаем словарь sites, содержащий координаты городов:
Moscow, London, Paris
Далее создаеми словарь distances.
это словарь словарей, где:
ключ первого уровня это город
ключ второго уровня это другой город

Расстояние вычисляем по формуле ((x1 - x2) ** 2 + (y1 - y2) ** 2) ** 0.5

Число π принять равным 3.1415926.
2. Определить, лежит ли точка внутри круга с центром в начале координат (0, 0) и радиусом 42.
Вывести True если точка внутри круга, в обратном случае False.
3. Аналогично проверить вторую точку point_2.
